# 🌧️ 소나기 디자인 시스템

**청량하고 시원한 느낌을 전달하는 디자인 시스템**

비가 내린 후의 맑고 깨끗한 공기, 물방울이 흐르는 듯한 부드러운 인터랙션을 디자인 언어로 표현합니다.

## 디자인 철학

### 청량감 (Freshness)
비가 내린 후의 맑고 깨끗한 느낌을 시각적으로 표현합니다.
- 시원한 블루-그레이 팔레트
- 깔끔한 여백과 공간 활용
- 높은 가독성과 명확한 계층 구조

### 유려함 (Fluidity)
물방울이 흐르는 듯한 부드럽고 자연스러운 인터랙션을 제공합니다.
- 부드러운 애니메이션과 전환 효과
- 유기적인 형태와 곡선 활용
- 직관적인 사용자 흐름

### 일관성 (Consistency)
시스템적 접근을 통해 예측 가능한 경험을 제공합니다.
- 명확한 디자인 토큰 정의
- 컴포넌트 간 일관된 패턴
- 체계적인 네이밍 규칙

## 시작하기

### 설치

```bash
npm install @sonagi/react @sonagi/tokens
# or
pnpm add @sonagi/react @sonagi/tokens
# or
yarn add @sonagi/react @sonagi/tokens
```

### 사용법

```tsx
import { Button } from '@sonagi/react';
import '@sonagi/react/styles';

function App() {
  return (
    <Button variant="primary" size="medium">
      Click me
    </Button>
  );
}
```
