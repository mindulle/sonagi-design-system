# 🌓 Theming Guide (테마 가이드)

Sonagi 디자인 시스템은 단일 브랜드에 국한되지 않고, 여러 독자적인 브랜드 및 서브 브랜드의 아이덴티티를 동시에 수용할 수 있는 **멀티 브랜드(Multi-Brand) 3-Tier 아키텍처**로 구현되었습니다.

---

## 1. 테마 종류

현재 제공하는 공식 테마는 다음과 같습니다:

### 1) Sonagi Core (기본값)
- **성격:** 감성적이고 따뜻한 느낌의 라이트/다크 테마 (조용한 깊이감).
- **디자인 특징:** Warm off-white 배경(`#faf9f7`), deep navy(`#1c2c4d`) 포인트 및 rain-blue(`#4A90E2`) 인터랙티브 강조색.
- **적용 대상:** 일반 콘텐츠 서비스, 홈페이지, 브랜딩 중심 웹.

### 2) The Desk Analyst (데이터 대시보드 테마)
- **성격:** 차갑고 정밀하며 밀도가 높은 기술 중심 데이터 분석 테마.
- **디자인 특징:** Clinical white 배경(`#FFFFFF`), 높은 줄 간격(`1.6`) 및 높은 대비의 블루(`#2563EB`) 강조색, 더 작고 촘촘한 코너 반경(`radius-md: 6px`).
- **적용 대상:** BI 대시보드, 어드민 패널, 데이터 스프레드시트 뷰.

---

## 2. 테마 적용 방법

테마는 HTML 문서 최상위 `<html>` (또는 원하는 스코프의 부모 컨테이너) 요소에 `data-theme` 속성을 부여하는 방식으로 쉽게 전환할 수 있습니다.

### 1) 라이트 모드 (기본값)
기본적으로 아무 속성도 부여하지 않거나 `data-theme="light"`를 지정합니다.
```html
<html data-theme="light">
  <!-- Sonagi Core Light 테마 적용 -->
</html>
```

### 2) 다크 모드
`data-theme="dark"` 속성을 설정합니다.
```html
<html data-theme="dark">
  <!-- Sonagi Core Dark 테마 적용 -->
</html>
```

### 3) Desk Analyst 테마 (데이터 대시보드 전용)
`data-theme="desk-analyst"` 속성을 설정합니다.
```html
<html data-theme="desk-analyst">
  <!-- Desk Analyst 테마가 적용되어 폰트 스케일 및 색상이 정밀 오버라이드됨 -->
</html>
```

---

## 3. JavaScript를 이용한 동적 테마 제어

```js
// 테마 동적 변경 함수
function setTheme(themeName) {
  if (themeName === 'light') {
    document.documentElement.removeAttribute('data-theme');
  } else {
    document.documentElement.setAttribute('data-theme', themeName);
  }
}

// 예: 다크 모드 토글
const isDark = document.documentElement.getAttribute('data-theme') === 'dark';
setTheme(isDark ? 'light' : 'dark');
```

---

## 4. 테마 오버라이드 개발자 규칙

새로운 브랜드 또는 하위 페이지를 위해 독자적인 테마를 추가해야 하는 경우:
1. `packages/tokens/tokens/themes/` 내에 `theme-새테마명.json` 포맷으로 신규 정의 파일을 생성합니다.
2. `semantics.json`에 정의된 정규 시맨틱 토큰 경로와 동일한 구조로 오버라이드할 값을 지정합니다. (예: `color.brand.primary`)
3. `pnpm build`를 수행하여 `variables.css` 내에 스코프 제한자(`[data-theme="새테마명"]`)를 포함하는 스타일이 자동 빌드되도록 합니다.
