## 4. Depth & Z-Index (그림자 및 심도)

### Shadows
- **Focus Ring (`var(--sng-shadow-focus)`):** 포커스 링 표시 전용 외곽 테두리
  - `Light:` `0 0 0 3px rgba(18,117,181,0.3)`
  - `Dark:` `0 0 0 3px rgba(25,145,185,0.4)`
- **Raised (`var(--sng-shadow-sm)`):** 카트, 패널 등 기본 띄움
  - `Light:` `0 1px 3px rgba(8,58,100,0.1)`
  - `Dark:` `0 1px 3px rgba(1,6,9,0.4)`
- **Floating (`var(--sng-shadow-md)`):** 드롭다운, 모달, 토스트
  - `Light:` `0 4px 12px rgba(8,58,100,0.12)`
  - `Dark:` `0 4px 12px rgba(1,6,9,0.5)`
- **Elevated (`var(--sng-shadow-lg)`):** 컨텍스트 메뉴, 모달 팝업 등 강한 띄움
  - `Light:` `0 8px 24px rgba(8,58,100,0.15)`
